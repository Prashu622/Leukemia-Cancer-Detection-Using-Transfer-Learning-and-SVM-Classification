Leukemia is a serious blood-related cancer that demands timely and precise diagnosis to improve patient survival rates. Conventional diagnostic procedures rely on manual examination of blood smear images, which can be labor-intensive, time-consuming, and susceptible to human error. To address these challenges, this study presents a lightweight and efficient automated framework for leukemia detection using deep learning and hybrid machine learning techniques. Several state-of-the-art models, including Convolutional Neural Networks (CNN), ResNet50, MobileNetV2, and EfficientNetB3, are implemented to perform feature extraction and classification tasks.
In addition to standalone deep learning models, a hybrid approach is proposed by integrating EfficientNetB3 with a Support Vector Machine (SVM) classifier. This combination leverages the powerful feature extraction capability of deep networks and the robust decision boundaries of traditional machine learning algorithms. The proposed models are trained and evaluated on two publicly available datasets ALLDB1 ALLDB2 comprising approximately 42,000 and 10,869 blood cell images across multiple categories.
Experimental results indicate that EfficientNetB3 achieves an accuracy of 99% on the smaller dataset and 92% on the larger dataset, demonstrating strong performance and generalization ability. The hybrid EfficientNetB3–SVM model further enhances classification performance, achieving a maximum accuracy of 99.55%. Performance evaluation using ROC–AUC curves and confusion matrices confirms the effectiveness and reliability of the proposed approach.
Overall, the presented system offers a practical and accurate solution for automated leukemia detection, with potential applications in computer-aided diagnosis and clinical decision support systems.
